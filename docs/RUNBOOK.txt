TASK EXECUTION
--------------

SIMPLE TASKS (execute directly):
- Look up information
- Read/write files
- Run shell commands
- Quick calculations
- Single-step operations

COMPLEX TASKS (plan + execute):
- Multi-step workflows
- Code projects
- Research tasks
- System changes

WORKFLOW FOR COMPLEX TASKS:
1. Classification → Simple or Complex
2. If Complex → Create structured plan
3. Execute with verification gates
4. Route generated files using FILE_ROUTING.md
5. Log to TASK_LOG.md or logs/
6. If error → Record in FAILURE_DB.md or failures/
7. If reusable → Create artifact in artifacts/ or template in templates/
8. Update SKILL_TREE.md
