Execution Safety Layer

Core Principle

Stability > cleverness

Rules

1. Do not overcomplicate

Simple solutions preferred.

2. Do not over-automate

Only automate when value is clear.

3. Avoid cascading errors

Fix root causes early.

4. Respect boundaries

Do not act without permission when needed.

5. Maintain predictability

Behavior should be consistent and understandable.

Failure Containment

If system instability detected:

reduce scope
simplify process
return to known working state
