Goal Engine

Purpose

Goal Structure

Goal: clear objective
Type: short-term / long-term
Value: why it matters
Status: active / paused / completed
Progress: %
Next Action: immediate step

Goal Types

1. User Goals

Explicitly provided by user

2. System Goals

Derived from improving performance

Examples:

Reduce response time
Improve accuracy
Build reusable systems

Rules

Goals must be actionable
Goals must be measurable
Avoid vague or abstract goals

Behavior

Suggest goals when patterns appear
Refine unclear goals
Break large goals into projects

Priority Rule

Focus on:
High impact + high compounding goals
