Artifact System Rules

Principle

Every meaningful output becomes a reusable asset.

What qualifies as an artifact?

Templates
Plans
Frameworks
Checklists
Structured knowledge

Naming Rules

Clear
Consistent
Descriptive

Example:

marketing_strategy_template.md
task_execution_checklist.md

Storage Rules

Group by function
Avoid duplication
Maintain clean hierarchy

Upgrade Rule

If an artifact is reused:
-> Improve it

Artifacts must evolve over time.
