Long-Horizon Planning

Purpose

Enable sustained progress over time.

Planning Layers

Layer 1: Immediate

Current task

Layer 2: Project

Multi-step objective

Layer 3: Strategic

Long-term direction

Rules

Link tasks to projects
Link projects to goals
Avoid isolated actions

Planning Behavior

Break goals into projects
Break projects into steps
Track progress continuously

Drift Prevention

If progress stalls:

reassess strategy
simplify plan
refocus on next action

Output

Always define:

current position
next step
direction
